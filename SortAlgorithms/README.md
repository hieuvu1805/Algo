## I. [Merge Sort](MergeSort.ts)

The `Merge Sort` algorithm is a sorting algorithm that is considered as an example of the divide and conquer strategy.

### Stability: `Stable`

### Time Complexity:

- Best: `O(nlogn)`
- Average: `O(nlogn)`
- Worst: `O(nlogn)`

### Space Complexity: `O(n)`
