# Algorithm

I use this project to practice algorithm and save my solutions

## How to run

1. Clone project
2. run `yarn` or `npm install` to install dependencies
3. run `yarn test` or `npm run test`

## Shortcode

- `TC`: Time Complexity
- `SC`: Space Complexity
